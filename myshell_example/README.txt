This is our readme.txt.

help: Displays the README.txt for the shell.
	example usage: help

quit: Exits out of the shell.
	example usage: quit

cd: Changes directory to given path.
	example usage: cd Downloads

pwd: Outputs the current directory to the console.
	example usage: pwd

echo: Outputs the argument given to the console.
	example usage: echo hello

ls: Prints the contents of the PWD
	example usage: ls

dir: Prints the contents of the given directory
	example usage: dir Documents

environ: Prints the environment variables
	example usage: environ

clr: Clears the terminal of all text
	example usage: clr
